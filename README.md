# gohtmx1

This is a test of bundling [htmx](https://htmx.org) and [Templ](https://templ.guide) with (Tailwind
and DaisyUI for CSS) into single binary written in Go.

## Running

After builing and running, open http://localhost:3000

```shell
$ make all
$ ./app
```

## Dev dependencies

* go
* bun
* npm (browserlist)
* make
